# echo-ent-example

Example application of Echo (Go web framework) and Entgo (ORM).

## Usage

Set $DSN for PostgreSQL connection string.

## Installation

```
$ git clone https://github.com/mattn/echo-ent-example
$ cd echo-ent-example
$ go build
```
